# Contributing

This repository is a compact public showcase of broader research-systems work.

## What Contributions Fit Best

- documentation improvements
- clearer architecture explanations
- small demo improvements that preserve the current public-safe scope
- tests that improve reliability without adding private dependencies

## What Should Not Be Added

- non-public datasets
- unpublished manuscript text
- institution-specific internal files
- secrets, tokens, cookies, or environment dumps
- copyrighted PDFs or restricted source documents

## Development Expectations

- keep the repository compact and readable
- preserve the review-gated, human-in-the-loop framing
- prefer deterministic sample data and inspectable outputs
- avoid overstating the system as fully autonomous research automation
